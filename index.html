What is Hakkuraifu ?

Hakkuraifu is a team created by Nazky (me).

In this site you can found every release made by the team, with the original creator and the code source of the project.

the main goal of hakkuraifu is to make available open source projects for the safety of users but also creators, here you can download different projects and have access to the source code to help creators or just check if the project is safety to use.
